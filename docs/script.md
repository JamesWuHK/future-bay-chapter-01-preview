# 剧集分场脚本版：时间档案城之《不被预测的人生》

## 1. 基本信息

- 对应章节：时间档案城样板章节
- 建议形态：35 到 45 分钟单集沉浸式在线剧情 / 互动叙事原型脚本
- 类型：未来科幻 / 悬疑追索 / 少年成长 / 文明议题戏剧
- 主剧情对标：《少数派报告》
- 命题气质参考：《千钧一发》

## 1.5 交付原则

这一版脚本默认服务于 `AI 在线沉浸式课程`，因此必须遵守三条硬原则：

1. 孩子全程留在故事世界内，不被任何真人老师或课堂口吻拉出情境。
2. 所有解释、追问、反馈和推进，都必须由世界内角色、系统界面或剧情事件完成。
3. 只有在本集完全结束后，才允许由固定 AI 形象数字导师做总结升华。

## 2. 故事核心

在“午夜归档”前的最后六小时，未来湾固定小队接下第一项正式灰银任务。12 岁女孩林澄拒绝签下自己的《最优人生路径确认书》，而你作为未来湾固定小队的新成员，必须带着祁野、闻夏、回响与阿潮进入时间档案城，决定要不要替一条还没被看见、却真实存在的未来争一个留下来的机会。

如果一个系统几乎能算准你最容易成功的人生，你还有没有权利去试那条不被看好的路。

你们一开始以为，这只是一场“孩子不想被安排人生”的争议。可越往里追，越发现时间档案城并没有把所有未来都摆出来。那些低概率、不好复制、却可能真的重要的“少数未来”，一直被压在更深的地方。

更危险的是，这不只是这一座城的问题，而是未来世界里越来越常见的一道裂口：

`它看起来像在帮你，其实正在一点点替你选路。`

## 3. 主要角色戏剧定位

### 林澄

主角。聪明、敏感、克制，不是爱闹脾气的孩子。她清楚那条被大家看好的路确实很强，也知道身边人是真的在替她着想，可她还是觉得，那不是她想活出来的人生。

### 档案守望者

引路人角色。像一位安静又可靠的长者，不替孩子做决定，只负责一次次把问题推回人自己的心里。

### 序列员 07

本集最重要的对立角色。他不是坏人，而是一个非常相信系统的人。他真心觉得，越早把大多数孩子送上更稳的路，城市就越少走弯路。只有他的立场够强，这场冲突才站得住。

### 风声婆婆

中段反转角色。她让故事从“孩子不想被安排”升级成“这座城是不是正在忘掉那些很慢、很难、却很重要的本事”。她给这座未来城市带来另一种温度和呼吸。

### 林澄的父亲

关系冲突角色。他爱女儿，也真心希望她过得稳一点。正因为如此，他说的话才会刺痛人，因为那正是很多真实大人会说出口的话。

## 3.5 角色说话风格锁定（本轮已更新）

- `阿潮`：短句、快、带劲，先报情况，再催出发。
- `祁野`：少废话，证据先行，不说虚的。
- `闻夏`：负责把复杂局面翻成人话，句子稳、清楚、能收场。
- `林澄`：就是孩子自己的嘴。直接说怕、不服、委屈、想试，不装深沉。
- `林澄父亲`：正常大人口气，出发点是担心，不讲大道理。
- `回响`：只说三件事，能做什么、会快多少、代价是什么。
- `守望者`：稳，但不神叨。负责把选择说清楚，不说谜语。
- `序列员 07`：冷静、利落、程序感强，但每句话都得让孩子听懂。
- `风声婆婆`：有画面感，但像真人聊天，不像神秘预言。
- `数字导师`：温柔、克制、收束，不说宣传片台词。

## 3.6 对白整改状态

- [x] 去掉孩子听不懂的抽象说法和拧巴句式。
- [x] 把孩子、大人、系统、长者的口气分开。
- [x] 把回响改成“说操作和代价”，不再说含混技术话。
- [x] 把守望者改成“把选择讲明白的人”，不再当谜语人。
- [ ] 进入正式配音前，再按口型长度做一轮收口。

## 4. 单集结构

建议按 `10 场世界内剧情 + 1 场课后收束` 推进。

整体节奏：

1. 立刻进入危机
2. 第一集就把固定团队立起来
3. 在追索中发现更大真相
4. 因错误与代价让冲突升级
5. 用“孩子自己签下责任”把高潮推到真正的承担
6. 用“未来湾持续冒险”而不是“单集结束”收束

## 4.5 趣味增强与实现状态

这一章的趣味，不是把一堆点子都硬塞进 current 主线，而是让世界自己会呼吸、角色自己会说话。为了避免文稿承诺跑在成片前面，这一节把 `已进 current`、`后补层` 和 `当前不承诺` 分开写清楚。

### 已经进入 current 主线的趣味层

- 阿潮会用短促播报把任务紧急感带进现场，让未来湾一开场就不是死板码头。
- 大厅里的礼仪辅助器只保留边角轻提示，作用是让孩子感到这座城“太会替人着想”，不是抢戏搞笑。
- 概率长廊会先把公开路径拍得真的诱人，再让林澄说出那种“我会的东西被挤成一小条”的不舒服。
- 暗层接口和沉默练习室已经把 `旧录音 / 风名标签 / 旧练习痕迹` 放进主线，趣味和温度都长在隐藏路径里。

### 已经写进文稿，但暂不算 current 完成

- `P05 / D03` 的荒诞微未来小窗还保留在文稿链里，但当前只算可选插片或点击后补，不算主线已经做完。
- 时间潮汐飞行器可以继续扩充更多冷幽默播报，不过当前主线只需要少量轮换句，先不把它做成整套秀场。
- 风名标签和短音频还能继续扩容，但 current 只保留最必要的一小层，让孩子知道这条路真的有人长期活过。

### 当前不进入主线承诺

- 巡检光束潜行这类会把节奏拉成小游戏的段落。
- 大量隐藏物件、收集册、成套彩蛋库。
- 会明显打断主冲突的整段轻喜剧支线。

### 当前的趣味原则

1. 先服务故事，再服务惊喜。
2. 趣味必须让孩子更懂角色，而不是更远离角色。
3. 趣味可以怪一点，但不能怪得像另一部片子。
4. 当前主线只保留最能抬高世界活性的那一层，其余统一放回后补层。

## 5. 分场脚本

### 第一场：灰银信号

- 场景地点：未来湾任务码头
- 时间：夜，临近潮汐换向时刻
- 场景功能：建立危机、倒计时、固定团队关系与正式任务入口

#### 画面

未来湾的海面并不是水，而是一层缓慢流动的光。港口上空悬着数百枚任务信号灯，大多安静柔和，只有一枚灰银色信号忽然急促闪烁。

你正站在任务台前，手腕上的未来湾固定小队权限环微微发亮。祁野已经把半透明数据板展开，闻夏正在把回响刚整理出的简报压成一页，阿潮则从高处掠过港口，边飞边播报。

#### 剧情推进

- 守望者以“正式灰银任务”而不是“临时帮忙”向你交付责任。
- 未来湾固定小队第一次同场亮相。
- 给出倒计时：距离午夜归档只剩六小时。
- 给出任务：不是替谁赢，而是确认一条人生有没有被完整看见。
- 任务简报先弹出四段争议现场切片，让孩子先看见未签署确认书、93% 匹配度、正在滚动的归档倒计时，以及林澄抬头去听那段系统没标出来的异常风声。
- 回响补充提醒：这次选择会进入固定小队责任档案。

#### 关键对白

档案守望者：
“这项灰银任务，现在由你来接。”

祁野：
“先看清，他们凭什么认定林澄只能走那条 93% 的路。”

闻夏：
“你拆数字，我去抢窗口。午夜一到，她连开口的地方都没了。”

回响：
“我是回响。帮你理线索，不替你下判断。”

“今晚这一单，会记进未来湾小队档案。”

档案守望者：
“从这一刻起，这不是旁观。”

“先去看清，这条人生到底有没有被完整看见。”

#### 场尾钩子

任务码头中央升起一张未签署的《最优人生路径确认书》，名字缓缓浮现：

`林澄，12岁。`

回响忽然又弹出一条异常提示：

`公开层只挂了三条路，可底下像是还有一条在动。`

### 第二场：紧急入港校验

- 场景地点：时间档案城外沿，入港校验幕前
- 时间：夜
- 场景功能：先把门槛、权限和倒计时讲清，不急着把孩子一下丢进大厅冲突

#### 关键对白

校验系统：
“争议档案已临时打开。先认清三件事：系统怎么给建议、资源会先往哪边走、午夜一到会发生什么。”

“请确认：你们现在能看，能问，能申请公开窗口。”

“但你们还不能替当事人改结果。”

祁野：
“先记住边界。我们现在只有观察权，没有改写权。”

闻夏：
“先进去。先把还能开的口子抢下来。”

### 第三场：悬浮档案大厅

- 场景地点：时间档案城，悬浮档案大厅
- 时间：紧接上一场
- 场景功能：立住三方站位，并让序列员 07 先合法赢走一小步；队友这时先不进画，只通过耳麦接线

#### 关键对白

林澄（看向我）：
“你也是来劝我的？”

“也想让我今晚把字签了？”

父亲（转向我）：
“我不是在逼她。我只是怕今晚一过，她以后想改都更难。”

序列员 07：
“我是序列员 07。今晚这份归档，由我负责。”

祁野（耳麦）：
“申请查看更底下那层对照记录。”

序列员 07：
“公开层你们能看。更底下的原始记录，今晚不开放。”

闻夏（耳麦）：
“他不是要赶我们走。”

“他是在卡时间，不让我们把话说完。”

### 第四场：等待区里的父女冲突

- 场景地点：档案大厅外侧等待区
- 时间：紧接上一场
- 场景功能：把“为你好”的压力压进更贴近现实的空间里

#### 关键对白

父亲：
“不是每个孩子都等得到这么高的匹配。它不是不让她试，是想先让她少摔几跤。”

林澄：
“那如果我偏偏想试的，就是那条你们都不放心的路呢？”

父亲：
“喜欢是一回事，往后一辈子怎么过，是另一回事。”

林澄：
“要是那条路越走越不像我，那还算是我的人生吗？”

### 第五场：概率长廊

- 场景地点：概率长廊
- 时间：夜更深
- 场景功能：把抽象议题变成可见的未来分支，制造第一次视觉奇观

#### 关键对白

孩子：
“为什么第三条路旁边写着‘现在还看不出能帮上多大忙’？”

序列员 07：
“因为这座城没法把还说不准的事，先当成大家都放心的路。”

林澄：
“可有些本事，本来就不是马上就能看出来的啊。”

#### 场尾钩子

长廊深处突然传来一段奇怪的风声，不像环境音，更像某种被删掉后还留下来的录音。

林澄猛地抬头：

“这不是长廊原本的声音。”

“像是从墙后面漏出来的。”

### 第六场：被藏起来的第四条路

- 场景地点：概率长廊暗层接口
- 时间：夜
- 场景功能：把“听见异常”推进成一条清楚的追索动作线，触发中段反转

#### 关键对白

林澄：
“这不是长廊原本的声音。它像是从里面漏出来的。”

祁野：
“波形不对。这不是回声，是被压在墙后的旧记录。”

回响：
“任务封签里的异常风声已调出。两段波形重合。”

阿潮：
“门缝在这儿。不是新路，是被折起来了。”

序列员 07：
“立刻停下。那不是公开比较层。”

孩子：
“既然它真的在，为什么连看都不让看？”

序列员 07：
“因为走这条的人太少，见效也慢。时间档案城现在还不敢把它直接挂出来。”

### 第七场：沉默练习室

- 场景地点：沉默练习室
- 时间：深夜
- 场景功能：让世界观获得温度，把抽象冲突转成人心冲突

#### 关键对白

风声婆婆（旧录音）：
“你听不见，不一定是因为它不存在。”

旧录音停了一拍，又补了一句：

“也可能是你前面太吵了。”

### 第八场：档案城的理由

- 场景地点：档案环下方，规则投影室
- 时间：距午夜不足五十分钟
- 场景功能：给反方最强陈词，避免故事沦为单边说教

#### 关键对白

序列员 07：
“你们现在盯着的是林澄一个人。”

“我盯着的是整座城这么多孩子。”

“今天把一条还没站稳的路往前提，别处能接住人的那份支持，就可能会少一点。”

孩子：
“可你们把第四条路藏起来，就已经先替别人做决定了。”

“你们先决定了，哪条路连看都不该看。”

序列员 07：
“你们现在觉得我太狠，是因为这份后果还没轮到你们来接。”

### 第八点五场：证据失效与团队分裂

- 场景地点：档案暗层出口与临时质疑预备区
- 时间：距午夜不足半小时
- 场景功能：让玩家因自己的选择付出代价，并把队伍逼到“只剩公开窗口还能争”的位置

#### 关键对白

回响：
“你要是同意，我可以拿以前像它的记录，先把第四路径这份材料补齐。”

“这样会快一点。”

“可到时候就分不清，哪些是原来就有的，哪些是我后来补上的。”

序列员 07：
“我不拦你们用回响。”

“可它后来补上的东西，不能当成最早那份证据。”

祁野：
“我刚就说了，别拿改过的材料去撞这道门。”

“这里最看重的，就是最早那份记录。”

闻夏：
“现在不是互相怪的时候。”

“私下补件已经来不及了。回响台的口子还在，我们只能去那里开口。”

林澄：
“要是这条路真被挂出来，他们会不会只是把我摆在那儿给人看？”

守望者：
“找到东西还不算完。真疼起来了，你们还肯不肯接着承担下去，这才算接了任务。”

### 第九场：自由回响台

- 场景地点：自由回响台
- 时间：午夜前最后四十分钟
- 场景功能：公开高潮，所有立场同时上桌，并把终局升级为“要不要真把后果承担下来”

#### 关键对白

林澄：
“我不是要别人都走这条路。”

“我只是想先看见它。为什么连这都不行？”

父亲：
“因为我怕你以后会怪今天的自己。”

序列员 07：
“这条路不是没有。”

“可现在不能先放到大家前面。”

“时间档案城得先顾住大多数，不能先替还没看清的孩子把路往前推。”

回响：
“按这里的规矩，我不能替任何人做这个决定。”

守望者：
“你现在不是在替她把以后一次定完。”

“你现在只要定一件事。”

“今晚，要不要先把这条路留下来。”

### 第十场：午夜前的选择与回航

- 场景地点：回响台与档案环核心
- 时间：午夜与任务结束后回航
- 场景功能：完成情感收束、结果落点与整季长线开篇

#### 关键对白

林澄：
“我不是非要谁先保证，我一定会赢。”

“我只是不想还没看清，就先被逼着把路选完。”

系统播报：
“第四路径，今晚暂时挂出。”

“今晚先公开给全城看。”

“这不是默认推荐，想走的人，还要继续观察。”

序列员 07：
“我只能替你开到这里。剩下的路，不再由系统替你决定。”

守望者：
“归档前最要紧的，不是哪条路看起来最安全。”

“是一个人能不能先看清，自己到底想怎么走。”

### 第十一场：课后 AI 数字导师总结

- 场景地点：未来湾回航后的收束界面
- 时间：任务完成后
- 场景功能：只在故事结束后做极简收束，不重新讲课，不把孩子拉出世界

#### 关键对白

数字导师：
“今天你们保住的，不是一条一定会赢的路。”

“你们保住的，是一个人先把路看清楚的机会。”

“以后你也会碰到这种时候。”

“有些事你现在还说不清，可你心里知道它很重要。”

“那你愿不愿意先替它留个位置，别那么早把它关掉？”

## 6. 本集的戏剧支点

为了保证后续改编时不丢失力度，这一集最重要的戏剧支点有七个：

1. 倒计时必须真实存在。
2. 序列员 07 必须足够有道理。
3. 父亲必须让人心疼也让人生气。
4. 玩家必须在中段做出一次会反噬剧情的选择。
5. 第四条路径必须具有真正的文明价值。
6. 结局不能是假胜利。
7. 趣味必须长在世界和角色里。

## 7. 在线沉浸产品适配建议

### 世界内引导如何替代老师讲解

- `档案守望者` 负责抛出问题，不负责讲道理。
- `序列员 07` 负责解释档案城的逻辑，让反方观点在剧情内成立。
- `系统界面` 负责展示数据、权重、锁定后果和路径差异。
- `风声婆婆` 负责把“难以打分的价值”变成孩子能感觉到的东西。
- `自由回响台` 负责承接孩子的表达和最终站位。

### 如果改成互动产品

- `P02-P06` 适合做探索式信息收集。
- `P09` 适合做“AI 加速整理是否授权”的代价节点。
- `P10-P11` 适合做公开辩论、责任签署与后果落地。
- `P12` 适合做极简收束与个人火种记录生成。
- 所有知识说明都应嵌在档案卡、路径图、授权弹层、系统提示和角色对白中。
- 不应插入剧情外老师讲解视频或课堂提问页。

### 课后 AI 数字导师的出现位置

- 只在第十场的世界内回航完成后出现。
- 不进入故事现场，不替林澄作决定。
- 不打断高潮前的情绪流动。
- 作用是总结、升华、连接现实，不是解释剧情给孩子听。

### 如果改成系列剧集

- 本集结尾可埋下序列员 07 自己那条“未公开分支”的伏笔。
- 林澄应从单集任务角色升级为固定成长伙伴起点。
- 风声婆婆可延展为后续世界层之间的隐秘连接者。
- “少数未来公开权”不必成为唯一长期主议题，但可作为“系统开始替人决定人生”这条长线第一次露面的裂口案例。
