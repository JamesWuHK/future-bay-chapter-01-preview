# 分镜脚本

## 1. 文档用途

这份文档用于把《不被预测的人生》这一集进一步转换成可供导演、分镜、动效、配音和前端共同使用的镜头级脚本。

它不是纯影视分镜，也不是纯交互说明，而是一份服务沉浸式在线课程的 `镜头 + 交互混合分镜脚本`。

## 2. 本章分镜总原则

### 2.1 第一人称为主

本章分镜默认采用 `第一人称主镜头`。

### 2.2 第三人称只作辅助

允许出现第三人称或半第三人称镜头，但只能用于以下几类时刻：

- 建立世界奇观
- 呈现复杂空间结构
- 展示林澄在巨大制度中的渺小感
- 强化高潮时的公共性

### 2.3 交互不应打断镜头情绪

交互触发最好长在镜头里，而不是视频播完再跳出去。

### 2.4 节奏规则

建议镜头节奏为：

- 开头稍慢，建立世界
- 中段逐步变快，进入调查与发现
- 暗层反转时节奏紧
- 沉默练习室再慢下来
- 辩论段清楚、稳、有压迫
- 结尾收束、安静

## 3. 编号规范

- `part_id` 使用 `P01` 到 `P12`
- `shot_id` 使用 `CH01-Pxx-Syy`

## 4. 整集可读性硬门

后续所有 part、beat、clip 都必须先过下面 5 个问题：

1. 这一段开始后 5 到 10 秒内，孩子能不能立刻知道 `我现在在哪里`。
2. 这一段结束前，孩子能不能说清 `谁正在要什么`。
3. 这一段的对白有没有先服务 `身份 / 动机 / 后果`，而不是先说抽象话。
4. 这一段和下一段之间，孩子能不能自然回答 `为什么接下来会到这里`。
5. 如果把这一段静音播放，孩子是否仍能看出这拍的大致戏剧职责。

## 5. 全片 Beat 合同总表

这张总表是当前页面镜像版分镜执行稿的核心部分。每个 beat 都必须同时回答六件事：`谁在场 / 在哪 / 谁说话 / 有没有旁白或系统声 / 现场听起来是什么 / 这一拍到底负责什么`。
