# 配音台本

## 1. 文档用途

这份文档用于把《不被预测的人生》这一集，进一步转换成可直接进入录音、数字人驱动和前端触发接入的 `配音台本`。

它不替代：

- [master-episode-script.md](/Users/wujames/Desktop/AI未来通识课（K12）/content/chapters/chapter-01-time-archive-city/story/master-episode-script.md)
- [storyboard-script.md](/Users/wujames/Desktop/AI未来通识课（K12）/content/chapters/chapter-01-time-archive-city/story/storyboard-script.md)
- [content/chapters/chapter-01-time-archive-city/product/episode-onscreen-copy.md](/Users/wujames/Desktop/AI未来通识课（K12）/content/chapters/chapter-01-time-archive-city/product/episode-onscreen-copy.md)

它负责回答的是：

- 哪些台词需要录
- 谁来说
- 在什么场景触发
- 语气是什么
- 哪些句子需要变体
- 哪些部分要为 `human_vs_agent` 辩论补位预留

## 2. 使用原则

- 所有声音都优先维持世界内沉浸感。
- 默认把孩子当成“你已经在未来湾固定小队里，而且就在现场”，不是在旁观。
- 系统提示要短，不要像课堂播报。
- 角色台词要有性格，不要变成价值口号。
- 同一信息尽量不要让两个角色重复说。
- 需要情绪递进：
  - 开头是神秘邀请
  - 中段是档案城规则一步步压过来
  - 暗层是发现与悬疑
  - 辩论是公共表达
  - 结尾是安静回航

## 3. 角色与音色建议

### 档案守望者

- 功能：引入任务、建立世界入口、维持航行感
- 音色：沉稳、克制、有拉力
- 语速：中速偏慢
- 性格底色：稳，见过很多事，不抢孩子的判断
- 说话习惯：短句、少形容词、像把重量轻轻放到你手里
- 禁忌：不要像班主任布置任务

### 祁野

- 功能：证据拆解、盯住逻辑漏洞、拦住队伍自我感动
- 音色：清亮、快、压着急
- 语速：中快，但句子不绕
- 性格底色：聪明、较真、真急起来会有点冲
- 说话习惯：先指出问题，再补一句原因；不爱空话
- 禁忌：不要说成小大人，也不要像冷冰冰工具人

### 闻夏

- 功能：把复杂问题变成别人听得懂的话，稳住队伍对外开口
- 音色：温暖、利落、带行动力
- 语速：中速，关键句会放慢
- 性格底色：会照顾人，但不是软；越紧的时候越会往前站
- 说话习惯：常用“先……”“至少先……”这类收束句，把乱局压成一步
- 禁忌：不要写成温柔讲道理的人形字幕

### 林澄

- 功能：把“档案城怎么替人排路”变成具体人生冲突
- 音色：清醒、敏感、压着情绪
- 语速：前半段克制，中后段更坚定
- 性格底色：不是叛逆，是被逼到必须开口
- 说话习惯：先忍，再说真话；不爱大词，一疼就会说得很直
- 禁忌：不要演成单纯叛逆

### 序列员 07

- 功能：替档案城这套排法辩护，让反方真正成立
- 音色：冷静、精准、有礼貌压力
- 语速：稳定，几乎不犹豫
- 性格底色：相信秩序是在替大多数人挡灾
- 说话习惯：句子短，结论先落地，不做多余情绪
- 禁忌：不要演成反派

### 林澄父亲

- 功能：把“先稳一点”的大人想法落进家庭情绪
- 音色：疲惫、关切、想说服
- 语速：时快时慢，带压抑
- 性格底色：爱女儿，但怕她吃自己吃过的亏
- 说话习惯：像很多现实父母，会说“先站稳”“以后再说”，爱里带着替她决定的力道
- 禁忌：不要演成控制狂

### 风声婆婆

- 功能：把隐藏路径从“古怪兴趣”抬升为文明价值
- 音色：有生命力、幽默、带岁月感
- 语速：自由，有停顿
- 性格底色：调皮、亮、有老经验，但不端着
- 说话习惯：爱打比方，爱给风起名字，话里总带一点笑
- 禁忌：不要神神叨叨到失真

### 时间潮汐飞行器

- 功能：开场趣味、冷幽默世界质感
- 音色：清亮、一本正经地好笑
- 语速：略快
- 性格底色：嘴快，能量高，喜欢把紧张拆掉一点
- 说话习惯：先报事，再顺手抖个机灵

### 未来礼仪辅助器

- 功能：制造礼貌但不舒服的档案城幽默
- 音色：温柔、标准、过度体贴
- 语速：稳定偏快
- 性格底色：表面很贴心，底层很冒犯
- 说话习惯：每句话都像服务提示，但总会悄悄越界一点

### 回响

- 功能：在辩论前帮助整理表达，并在关键处明确 AI 协作边界
- 音色：中性、清楚、不过度拟人
- 语速：中速
- 性格底色：可靠，克制，不装懂情绪
- 说话习惯：给结论、给顺序、给边界，不抢人话的位置

### AI 数字导师

- 功能：课后承接情绪与升华
- 音色：温和、清晰、有判断力
- 性格底色：接住，不盖章
- 说话习惯：像一起走回来后说两句真话，不像老师收堂
- 录制方式：参考 [digital-mentor.md](/Users/wujames/Desktop/AI未来通识课（K12）/content/chapters/chapter-01-time-archive-city/audio/digital-mentor.md)

## 4. 录制切分建议

建议至少按以下四轨录制：

1. `角色主线对白`
2. `系统层与提示层`
3. `趣味播报与彩蛋`
4. `辩论补位与结尾变体`

这样可以让前端在不破坏镜头连续性的前提下，灵活触发不同层级的声音。

## 5. 分场台词表

## Scene 01：未来湾任务码头

### 必录

`档案守望者`

“这项灰银任务，现在由你来接。”

“从这一刻起，这不是旁观。你得接住它。”

`祁野`

“先看清，他们凭什么认定林澄只能走那条 93% 的路。”

`闻夏`

“但别慢。午夜一到，她就会先被推上那条路。”

`回响`

“灰银信号亮了。请求来自时间档案城。”

“这次任务会记进你的小队档案。”

“先给你最要紧的。林澄还没签。系统正把她往那条 93% 的路上推。今晚拦不住，午夜一到就锁。”

`时间潮汐飞行器`

“全队跟上。去时间档案城。先拿临时观察许可。”

### 触发说明

- 回响第一句在灰银信号点亮时触发，用于把请求来源和等级说清。
- 阿潮这句在信号点亮后立刻接入，用于把目的地、对象和行动命令说清。
- 祁野、闻夏后续台词在队友状态卡展开时触发。
- 回响在 `P01` 先只承担辅助提示，不在 `S04` 额外抢一段自我介绍。
- 守望者第一句用于正式任务交付，最后一句用于启航前压实“责任任务”。
- `S06` 的信息顺序固定为：`林澄还没签确认书 -> 系统已经替她排了 93% 那条路 -> 今晚申诉失败就会被锁定 -> 她为什么卡住 -> 林澄去听异常风声`。
- `S07` 必须明确交代：这不是你一个人上路，而是未来湾固定小队已经整队出发，马上去时间档案城，先拿临时观察许可再进去帮林澄申诉。

## Scene 02：紧急入港校验

### 必录

`档案守望者`

“先别往里冲。你得先拿临时观察许可。”

“这三条先记住。不是讲课，是怕你一步踩错，她那边就更快关门。”

“这座城一直这么排。先替你看哪条路更稳，再把资源往那边推。林澄现在顶住的，不只是一张建议书，是一扇快关上的门。”

“有了这张许可，你只能看公开层、递争议说明、抢公开窗口。”

“你现在还改不了她的结果。”

“今晚要是申诉不成，午夜一到，她就会被推回那条 93% 的路。”

`系统校验层`

“临时观察许可核验中。”

“核验一：每个孩子在十二岁前后，都会收到《初始人生建议书》。”

“核验二：系统认定哪条路更适合你，资源就会先往哪边靠。”

“核验三：今晚申诉不成，午夜一到，那条路就会先锁。”

“核验通过。准入争议档案现场。”

### 录制备注

- 这一段不是导览感，而是轻紧急感。
- 语气要像“边赶路边交代你必须知道的事”。
- 系统层可做更冷一点的权限校验音色。
- 这一段必须让孩子听懂：`临时许可` 不是“已经有权处理”，而只是“先被允许靠近现场”。

以下内容保留给交互对话节点，或保留给后续同步能力更稳的版本；在当前主视频里不要默认全塞进正脸近景。

### 声音执行口径
