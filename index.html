W10-W11 成片护栏

正式视频判定

硬规则 `单张照片推拉 / scene still 呼吸动效 / clean plate + 轻 motion / 主要靠离屏声音撑住的静态 hold`，都不再被视为 chapter-01 的正式视频完成态。

  • 这些位只允许作为 `paper cut`、`temporary bridge`、`voice-led hold` 或 `replacement debt`
  • 一旦目标是 `current delivery / final / 完整一集`，它们默认必须进入替换或收紧队列
旧资产复用顺序

默认判断 不浪费已有真实 clip，但复用必须服从故事与连续性，不再为了省 token 把不该留的旧镜头硬塞进成片。

  • 优先顺序固定为：`TRIM -> REORDER -> BRIDGE -> same-space upgrade -> REDO`
  • accepted clip、accepted_with_notes clip、可升格 bridge clip 与局部可用 rescue clip，都应先盘点再决定是否重跑
人物一致性门

最高视觉门之一 林澄、祁野、闻夏、序列员 07、守望者等识别核心人物，默认必须继续沿用 `canonical portrait + actor refs + scene-state keyframe`。

  • 相邻镜头如果同一角色明显像两个人，直接判连续性失败
  • face swap 只允许作为 `recovery path` 评估，不是默认主路线
剧情过渡门

儿童可跟上 每一处 shot-to-shot、part-to-part 过渡，都必须能回答“上一拍改变了什么，为什么下一拍会出现在这里”。

  • 如果观感像“换页面 / 换模块 / 预告片拼贴”，默认先判为装配失败
  • 此时优先回 `editor + directors-room`，在 `TRIM / REORDER / BRIDGE / REDO` 中给出单一结论
桥接优先原则

不要动不动整段重开 高潮、转折、回航如果已经基本讲通，后续先找缺失的 `consequence bridge / reaction bridge / handoff bridge`。

  • 能用桥接和重剪解决，就不要浪费已成立的真实 clip
  • `P10 -> P11` 当前就是这一原则下的重点复审位
交付资格

最终优先级 chapter-01 的目标不是把零散成功资产并排展示，而是形成一条孩子能顺着看下去的完整故事视频。

  • 最终装配优先级固定为：`故事因果清楚 -> 人物形象稳定 -> 过渡合理 -> 再考虑资产复用率`
  • 只有符合这条顺序的版本，才有资格升格为更正式的 `current` 或 chapter-level master
